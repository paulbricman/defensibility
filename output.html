Textual-Symbolic Interoperability by Unsupervised Machine Translation

Language models excel at natural language, yet their ability to employ proper logical form is often crippled by their conversational fluency. In this work, we bootstrap textual-symbolic interop using unsupervised machine translation, building towards sound machine reasoning.

  1. Introduction

    This work is embedded in an agenda centered around adversarial reflection. Computational argumentation provides principled means of evaluating debates, yet often requires natural language utterances be abstracted in symbolic notation. Other debate evaluation methods might require compound natural language utterances be first decomposed into atomic ones. In addition, unsupervised epistemic interpretability methods might benefit from natural language exemplars related to each other in various formal ways. Given these prospects, we aim to enable reliable interoperability between formal and natural languages.

  2. Method

    We cast textual-symbolic interop as machine translation, and employ unsupervised learning techniques to exploit the symmetries of the task. Starting with a seed dataset of linked representations, we few-shot prompt a language model to produce novel data points. We then parameter-efficient fine-tune the model on synthetic data points filtered for backtranslation consistency, and repeat. We finally analyze the resulting dataset using an array of metrics.

  4. Discussion

    Overall, unsupervised machine translation appears to provide reliable tooling for debate evaluation and epistemic interpretability. However, it remains to be seen whether the limited means of representing uncertainty might impair downstream performance.
